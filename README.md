### 버전 안내 
- V.1.0 - 급식 확인 위젯 개발

- V.1.1 - 급식 시간별 확인

- V.2.0 - 급식 확인표 알러지 정보 오류 수정
- V.2.1 - 내일 급식 확인 및 정규 표현식 최적화
- V.2.2 - 급식 확인표 '/' 포함 시 발생하던 오류 수정 (2021. 08. 01)
- V.2.2.1 - 오늘의 급식 추가 (2021. 8. 21)

- V.3 - IOS15 최적화 (2022. 3. 3)
- V.4 - 학교 사이트 주소 이전에 대처한 업데이트, 업데이트 자동화 기능 추가
